# PromptPlot Examples

This directory contains comprehensive examples and tutorials for using PromptPlot v2.0.

## Directory Structure

```
examples/
├── README.md                    # This file
├── basic/                       # Basic usage examples
├── advanced/                    # Advanced features and workflows
├── file_conversion/             # File conversion examples
├── configuration/               # Configuration examples
├── integration/                 # Integration examples (API, CLI, etc.)
├── sample_files/                # Sample files for testing
├── tutorials/                   # Step-by-step tutorials
└── performance/                 # Performance optimization examples
```

## Quick Start

1. **Basic Drawing**: Start with `basic/simple_drawing.py`
2. **File Conversion**: Try `file_conversion/svg_to_gcode.py`
3. **Configuration**: See `configuration/custom_settings.py`
4. **Tutorials**: Follow `tutorials/getting_started.md`

## Running Examples

Most examples can be run directly:

```bash
cd examples/basic
python simple_drawing.py
```

Some examples require additional setup or configuration. Check the individual README files in each directory for specific requirements.

## Sample Files

The `sample_files/` directory contains various file formats for testing:

- **SVG files**: Vector graphics for conversion
- **G-code files**: Direct G-code for plotting
- **DXF files**: CAD drawings
- **Configuration files**: Example configurations
