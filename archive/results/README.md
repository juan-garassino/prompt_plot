# Results Directory

This directory contains outputs and results from PromptPlot v2.0 operations.

## Structure

- `gcode/` - Generated G-code files
- `images/` - Captured images and visual outputs  
- `visualizations/` - Generated visualization files (PNG, HTML, etc.)
- `reports/` - Drawing session reports and summaries
- `logs/` - Execution logs and debugging information

## File Organization

Files are organized by date and workflow type to maintain clear separation
of different drawing sessions and experiments.
