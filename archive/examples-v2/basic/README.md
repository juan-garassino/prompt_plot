# Basic Examples

This directory contains simple examples to get you started with PromptPlot.

## Examples

### simple_drawing.py
Basic drawing with simulated plotter. Good starting point for new users.

**Features:**
- Simulated plotter setup
- Simple prompt execution
- Basic error handling

**Usage:**
```bash
python simple_drawing.py
```

### hardware_connection.py
Connect to real plotter hardware.

**Features:**
- Serial plotter connection
- Connection testing
- Basic hardware interaction

**Requirements:**
- Physical plotter connected via USB/Serial
- Correct port configuration

**Usage:**
```bash
python hardware_connection.py
```

### multiple_shapes.py
Draw multiple shapes in sequence.

**Features:**
- Multiple drawing commands
- Shape positioning
- Coordinate management

### workflow_comparison.py
Compare different workflow types.

**Features:**
- Simple vs Advanced workflows
- Performance comparison
- Feature demonstration

## Prerequisites

- PromptPlot v2.0 installed
- Python 3.8 or higher
- For hardware examples: physical plotter

## Configuration

Most basic examples use default settings. For custom configuration, see the `../configuration/` directory.
